# @rule-io/vendor-samfora

Samfora preset for the Rule.io SDK. Donation-platform automation flows for [Samfora](https://samfora.org). Default copy ships in Swedish.

```ts
import { samforaPreset, SAMFORA_FIELDS, SAMFORA_TAGS } from '@rule-io/vendor-samfora';
```

See the [main `@rule-io/sdk` README](../rule-io-sdk/README.md) for full SDK docs.
